In todays world the web plays a vital role in a variety of complex and sophisticated industry applications. The web application continue to contribute to problems of usability, quality, reliability etc. With number of changes and advancements in the characteristics of web applications certain new methods needs to be introduced.Thus web engineering mainly focuses on the methods, and techniques that are considered as the main foundation of web application. Web engineering focuses on the methodologies, techniques, and tools that are the foundation of Web application development and which support their design, development, evolution, and evaluation. Web application development has certain characteristics that make it different from traditional software, information system, or computer application development.

Web Engineering is multidisciplinary and encompasses contributions from diverse areas: systems analysis and design, software engineering, hypermedia/hypertext engineering, requirements engineering, human-computer interaction, user interface, information engineering, information indexing and retrieval, testing, modelling and simulation, project management, and graphic design and presentation. Web engineering is neither a clone nor a subset of software engineering, although both involve programming and software development. While Web Engineering uses software engineering principles, it encompasses new approaches, methodologies, tools, techniques, and guidelines to meet the unique requirements of Web-based applications.
